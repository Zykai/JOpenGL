# JOpenGL
- Ein Programm, das als Startpunkt für OpenGL-Applikationen gedacht ist
- enthält eine Shader-Klasse
- Benutzt [LWJGL 3](https://github.com/LWJGL/lwjgl3) und [JOML](https://github.com/JOML-CI/JOML) über Gradle
